    Ich habe auch schon einige Webseiten mit und ohne Bootstrap erstellt. Hier habe ich zwei dieser Webseiten vorgestellt.

    Webseite mit Bootstrap und Server-Verbindung

    Diese Webseite wurde mit HTML, JavaScript und dem Bootstrap-Framework kreiert und auf dieser Webseite gibt es auch eine Verbindung mit einem Server dem Daten von einem Formular übergeben werden.
    Dieser Server muss erst gestartet werden, aber die Webseite funktioniert auch ohne den Server.

    Um den Server zu starten:
    1. In CMD zum Dateipfad skiserver\server navigieren.
    2. "npm update" Befehl eingeben
    3. "npm start" Befehl eingeben
